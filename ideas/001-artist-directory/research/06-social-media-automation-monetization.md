# Research: Social Media Automation — Monetization Impact (Supplementary)

**Date:** 2026-04-10
**Source:** Perplexity `sonar-reasoning-pro`
**Status:** Future consideration — not a launch feature

---

## Summary

If automated social media promotion (done-for-you, not tools) were bundled into an artist subscription tier, it could roughly double the artist-side revenue projection. This is filed as supplementary research for future reference.

## Key Numbers

- **Conversion lift:** Baseline 10% → estimated 14-16% with social promo as tier differentiator
- **Churn reduction:** Baseline ~5%/mo → estimated 3.5-4%/mo (promotion creates switching cost)
- **Revenue impact:** ~$17,760/yr → ~$30,000-45,000/yr (net of COGS)
- **Per-artist COGS:** $2-5/month (AI content generation + API costs across platforms)
- **Margin:** 67-87% gross on the social component (within healthy SaaS range)

## Critical Caveats

1. "Done for you" framing is essential — "here are tools" doesn't move the needle
2. Must demonstrate visible results (monthly impressions/engagement reports) or stickiness evaporates
3. Should be a tier differentiator, not a standalone product
4. AI content quality must be genuinely good — spammy posts damage platform reputation
5. Requires significant development investment to productize (separate from directory platform)

## Why This Is Deferred

The social media automation tool (PolyWiz) is currently an internal Polymash tool on Airtable infrastructure. Productizing it for external use requires:
- Supabase rebuild
- Monetization and tiering features
- Broader market positioning (not art-specific — works for podcasts, blogs, any business)
- Its own development trajectory independent of the directory platform

The directory platform should validate its core value proposition (exhibition matching, discovery, lead generation) before adding this layer.
